WHAT IS DEVGORDON?
================================================================================

DevGordon is a conversational AI agent that:
1. Runs entirely on your machine (no cloud APIs)
2. Generates infrastructure code (Ansible, kubectl)
3. SCANS its own generated code for security issues BEFORE execution
4. Waits for user approval before running anything
5. Integrates with your local Kubernetes, Jenkins, and Docker

Key Innovation: Agent self-checks code quality via SonarQube standards.
Most AI agents generate and hope. DevGordon verifies first.
